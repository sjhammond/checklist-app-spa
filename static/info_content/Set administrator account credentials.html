2B. Set administrator account credentials

XProtect VMS must access the device with administrator-level access. You may use the built-in administrator account, if the device comes with one, or create a new one.

For security reasons, Milestone recommends you change any default passwords before deploying the camera, even if you will be deploying the camera to an isolated camera network that is reachable only by the recording server (recommended).

For select devices, you may optionally set the administrator account credentials in the device at the time you add it to XProtect VMS, in which case you can choose to skip this step. Refer to the release note on the Milestone website > Support > Download Software > Release Notes to verify whether this functionality is supported for your device.
