26D. Configure performance settings

Configure the Mobile server performance settings to meet any requirements in the design specification on the Servers > Mobile Servers > Performance tab.

If the installation uses multiple live streams for adaptive streaming, make sure to select the adaptive streaming mode strategy the server should use. If a stream uses privacy masking, or a Mobile or Web client fails to receive the direct stream, or if you disable direct streaming, theThe Mobile server will transcodes the video into MJPEG format. Use the transcoded image stream limitations settings to optimize the performance of the Mobile server when using transcoded streams.
