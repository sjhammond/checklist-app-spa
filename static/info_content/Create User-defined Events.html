20A. Create User-defined Events

You can use user-defined events to allow users to manually trigger a rule. User-defined events are also exposed in the MIP-SDK, allowing third-party applications and integrations to trigger rules in XProtect VMS.

Create any User-defined Events required for the installation on the Rules and Events > User-defined Events node.
