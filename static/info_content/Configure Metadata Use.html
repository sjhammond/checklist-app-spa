19E. Configure Metadata Use

Enable metadata search visibility for Smart Client users under the Metadata Use node

Note: Users assigned to roles with permissions to access metadata will see the enabled metadata search categories on the Smart Client search tab.

Note: For results to be available to Smart Client users to search, hardware devices supporting the Milestone Metadata Search categories must be configured to deliver such metadata and the metadata streams must be enabled for the device in XPprotect VMS under Devices > Metadata.

See also: 16. Configure metadata, 23. Configure users and security
